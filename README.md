# euchre-ts
EuchreTs is the complete euchre card game written purley in typescript.

### Installation
Setup:

```sh
git clone https://github.com/suite/euchrets.git
cd euchrets
npm i # "sudo npm i" if you're on macOS or Linux
```

Run After Setup:

```sh
npm start
```
